# リポジトリ URL メモ

本プロジェクトの **GitHub リポジトリ**は次のとおりです。

| 種別 | URL |
|------|-----|
| ブラウザで開く | [https://github.com/nakaokazuki/blackjack](https://github.com/nakaokazuki/blackjack) |
| HTTPS（`clone` / `remote`） | `https://github.com/nakaokazuki/blackjack.git` |
| SSH | `git@github.com:nakaokazuki/blackjack.git` |

`git clone` の例:

```bash
git clone https://github.com/nakaokazuki/blackjack.git
```

リモートを追加する例:

```bash
git remote add origin https://github.com/nakaokazuki/blackjack.git
```
